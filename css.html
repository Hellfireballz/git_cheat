CSS Cheat Sheet

This playful fox is here to help you master CSS! Dive into the commands below:

CSS Properties and Values

Property Description
color Sets the text color of an element.
background-color Sets the background color of an element.
font-size Sets the size of the font.
font-family Specifies the font for text.
margin Sets the space around an element.
padding Sets the space inside an element, between the content and the border.
border Specifies the style, width, and color of an element's border.
width Sets the width of an element.
height Sets the height of an element.
display Specifies the display behavior of an element (e.g., block, inline).
flex Specifies how a flex item will grow or shrink in relation to other items.
justify-content Aligns flex items along the main axis of a container.
align-items Aligns flex items along the cross axis of a container.
position Specifies the type of positioning method used for an element (e.g., static, relative, absolute).
top, right, bottom, left Specifies the position of an element relative to its positioned ancestor.
z-index Sets the stack order of an element (higher values are on top).
opacity Sets the transparency level of an element.
