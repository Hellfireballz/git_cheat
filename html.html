Mischievous Rabbit's HTML Cheat Sheet

This mischevious rabbit knows how to make HTML fun! Learn the essentials below:

HTML Commands

Tag Description
<html> Defines the root of an HTML document.
<head> Contains metadata/information for the document (like title, link, meta).
<title> Sets the title of the webpage, shown in the browser tab.
<meta> Defines metadata like character set, description, or author of the document.
<body> Contains the content of the HTML document (like text, images, and links).
<h1> - <h6> Defines HTML headings. <h1> is the largest, and <h6> is the smallest.
<p> Defines a paragraph of text.
<a href="URL"> Creates a hyperlink to a URL or page.
<img src="URL" alt="description"> Embeds an image in the webpage.
<div> Defines a section or block in an HTML document.
<span> Defines a small section or inline block of text.
<br> Inserts a line break in the text.
<ul> Creates an unordered (bulleted) list. Items are defined with <li>.
<ol> Creates an ordered (numbered) list. Items are defined with <li>.
<form> Defines an HTML form for user input, often paired with <input> fields.
<input type="text"> Defines an input field for user text input.
<button> Defines a clickable button.
<table> Defines a table. Table rows are defined with <tr>, headers with <th>, and data cells with <td>.
<footer> Defines the footer section of a page.
