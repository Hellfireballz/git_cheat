Samurai Goose's VS Code Cheat Sheet

The samurai goose is here to guide you through your VS Code journey! Stay sharp with these commands:

VS Code Shortcuts and Commands

Command Description
Ctrl + P Quickly open any file by name.
Ctrl + Shift + P Open the Command Palette to search and run VS Code commands.
Ctrl + B Toggle the sidebar visibility (Explorer, Source Control, etc.).
Ctrl + ` Toggle the integrated terminal.
Ctrl + K, Ctrl + S Open the Keyboard Shortcuts pane.
Ctrl + / Comment/uncomment the selected lines of code.
Alt + Click Insert additional cursors for multi-line editing.
Ctrl + Shift + F Search across all files in the workspace.
F2 Rename the symbol under the cursor.
Ctrl + Shift + E Focus on the Explorer pane.
Ctrl + Shift + M Open the Problems pane to view errors and warnings.
Ctrl + Shift + X Open the Extensions Marketplace.
Ctrl + K, Z Enter Zen Mode (distraction-free mode).
Ctrl + D Select the next occurrence of the current word for editing.
Ctrl + Shift + [ Fold the current code block.
Ctrl + Shift + ] Unfold the current code block.
